# Breadcrumb
## Purpose
A component to be used for navigation.



## Usage

#### Basic usage

```jsx
<Breadcrumbs
   
   activeItem={2}
    routes = {[
  	{
    	href: "/introduction",
      name: "introduction"
  	},
  	{
    	href: "/introduction/pricing",
      name: "pricing"
  	},
  	{
    	href: "/introduction/pricing/general",
      name: "general"
  	}
  ] }
  
/>
```
